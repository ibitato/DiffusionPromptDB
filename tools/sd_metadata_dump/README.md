# Stable Diffusion Metadata Exporter

Utility created to replicate the "PNG Info" feature without touching the main
project tree. It scans any folder that contains Stable Diffusion PNG images
generated by StabilityMatrix / AUTOMATIC1111 and produces a structured export
with the positive prompt, negative prompt and execution settings of every
image.

## Usage

```bash
# Run from the repository root
python3 tools/sd_metadata_dump/export_sd_metadata.py \
  --source /mnt/d/sd-matrix/Data/Images \
  --output tools/sd_metadata_dump/output/metadata.jsonl \
  --format jsonl
```

- `--source` points to the folder that contains the `Text2Img`, `Img2Img`, etc.
- `--output` is the path to the JSONL or CSV file that will be created.
- `--format` can be `jsonl` (default) or `csv`.

The JSONL output provides one JSON object per line:

```json
{
  "relative_path": "Text2Img/2025-11-12/00001-3710984639.png",
  "positive_prompt": "masterpiece, best quality, ...",
  "negative_prompt": "XL_NEG-neg",
  "settings": {
    "Steps": "20",
    "Sampler": "DPM++ 2M SDE",
    "Seed": "3710984639",
    "Model": "pornmaster_proSDXLV7"
  },
  "raw_parameters": "full original chunk content"
}
```

The CSV output flattens the structure and stores the `settings` dictionary as a
JSON string in the `settings_json` column for easy spreadsheet filtering.

No dependencies are required beyond the Python standard library.
